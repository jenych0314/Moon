# 출력
## 1. document.getElementById().innerHTML
```html
<body>
    <p id="test"></p>
    <script>
        //id로 html요소(elem) 접근
        let elem = document.getElementById("test");
        // elem의 innerHTML 속성을 통해 내용 적용
        elem.innerHTML = "10" + 1;
    </script>
</body>
```
