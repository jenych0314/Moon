# Q1. 어떻게 로그인을 유지할 수 있을까?
웹 클라이언트에서 웹 서버에 정보를 1회 건네주고 만다
그런데 어떻게 같은 웹 사이트에서 로그인을 유지할 수 있는 것일까?

# 쿠키
웹 서버에 요청을 보낼 때 같이 보내는 작은 정보

# 세션
웹 서버에 저장되는 정보

set-cookie
쿠키가 클라이언트 측 저장에 위험 -> 클라이언트가 변경 가능해서
서버에 쿠키 저장 -> 세션
세션id -> 쿠키
세션id 유추할 수 없는 일련의 문구로
세션id 탈취
